Rebel Pilot

Created by Hero Nguav
Editing by Howard Day

A remarkably complex model, this Rebel pilot includes the facility to move or rotate the arms, legs and head independently into almost any posture. The colour of the flightsuit and visor can be redefined, and the file includes definitions for both male and female pilots. This one may take a little work to incorporate into a scene, but it looks excellent.

Original POV-Ray model by Howard Day

X-Wing Fighter

Original POV-Ray model by Dylan Beattie

The famous Incom T65 Space Superiority fighter, first seen in Star Wars: A New Hope. This model was written completely in POV-Ray, taking measurements from the AMT plastic model kit; the markings and textures are derived from various photographs and stills from the film. The file includes the ability to change the wing rotation angle and the squadron colour using variable definitions; see the documentation in the ZIP package for details.

Download ZIP file (120K)
