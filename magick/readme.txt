This file contains a complete set of DLLs taken from the ImageMagick-5.3.0 distribution (http://www.imagemagick.org) for Win2K.

These must be extracted to the perl/bin directory and used in conjunction with PerlMagick-5.3.0.zip to install PerlMagick. For more info see http://www.zepler.org/~dylan/magick-win2k/
