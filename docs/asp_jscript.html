Ten Reasons Why You Should Be Using JScript Instead Of VBScript

VBScript is an interpreted scripting language, commonly used for writing websites based on Microsoft's Active Server Pages technology. Loosely based on the syntax of the popular Visual Basic programming langugage, VBScript is the de facto standard for ASP web development, probably due to the vast number of ASP books, tutorials and examples which use VBScript to teach and demonstrate principles of ASP programming.

JScript is the Microsoft implementation of the ECMA 262 language specification (ECMAScript Edition 3). With only a few minor exceptions (to maintain backwards compatibility), JScript is a full implementation of the ECMA standard.
